title: Forgot Password
description: How to set up forgot password flow using Authverse.
---

Authverse streamlines the addition of a "Forgot Password" flow with a single command.

## Installation

Execute the following command:

```package-install
npx authverse@latest forget
```

This command automatically:

- **Injects** password reset logic.
- **Creates** the email handler.
- **Sets up** the API route.
- **Adds** required environment variables.
- **Configures** the Resend email template.

## Environment Configuration

After the command completes, add the following variables to your `.env` file:

```bash
RESEND_API_KEY=
EMAIL_SENDER_NAME=
EMAIL_SENDER_ADDRESS=
```
